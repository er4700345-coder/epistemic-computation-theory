# ECT Research Roadmap

This document outlines the planned trajectory of the Epistemic Computation Theory research program.

---

## Phase 1 — Foundation (Current)

**Goal:** Establish the core theoretical skeleton.

- [x] ECT-1 Conjecture formally stated
- [x] ECT-2 Conjecture introduced
- [x] ECT-3 Conjecture introduced
- [x] Core definitions published
- [x] Relationship to Gödel, Halting Problem, Kolmogorov documented
- [x] GitHub repository live
- [ ] ECT-1 proof sketch with supporting lemmas
- [ ] Formal treatment of ECT-2
- [ ] Formal treatment of ECT-3
- [ ] LaTeX preprint v0.1 submitted to arXiv

---

## Phase 2 — Formalization

**Goal:** Turn conjectures into a rigorous mathematical framework.

- [ ] Define complexity class `EK` (Epistemically Knowable problems)
- [ ] Define complexity class `EU` (Epistemically Unknowable problems)
- [ ] Prove or disprove: `EK ⊂ decidable`
- [ ] Formalize Epistemic Collapse as a computational event
- [ ] Establish the Observer Context algebra for ECT-3
- [ ] Peer review outreach (Scott Aaronson, complexity theory community)
- [ ] LaTeX preprint v0.2

---

## Phase 3 — Language Integration

**Goal:** Implement ECT semantics in the SLIME programming language.

- [ ] Design ECT type system primitives:
  - `uncertain<T>`
  - `contradictory<T>`
  - `collapsed<T>`
  - `observer<T, O>`
- [ ] Implement type checker for ECT types in SLIME compiler
- [ ] Write ECT type system specification document
- [ ] Example programs demonstrating epistemic type behavior
- [ ] SLIME + ECT joint paper

---

## Phase 4 — ECT-AI

**Goal:** Build a reasoning engine grounded in ECT semantics.

- [ ] Design ECT-AI architecture
- [ ] Implement a reasoning engine that tracks its own epistemic boundary
- [ ] Benchmark against standard AI reasoning baselines
- [ ] ECT-AI can formally output: `I cannot know this` with a proof
- [ ] Research paper: *ECT-AI: A Reasoning Engine That Knows What It Cannot Know*

---

## Phase 5 — Ecosystem

**Goal:** Make ECT a recognized research program.

- [ ] Lab website: `ect.divinesunday.dev`
- [ ] ECT newsletter / research log on Medium
- [ ] Conference submissions
- [ ] Open ECT to external contributors
- [ ] ECT v1.0 — stable theoretical foundation declared

---

## Long-Term Vision

ECT becomes the foundational framework for reasoning about AI epistemics — a standard lens through which AI systems are evaluated not just on what they know, but on what they can formally prove they cannot know.
