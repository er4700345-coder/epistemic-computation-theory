# Contributing to ECT

ECT is an open research framework. Contributions are welcome across theory, formalization, and implementation.

## Ways to Contribute

### 🔬 Theory
- Propose new conjectures or counterexamples
- Formalize existing definitions more rigorously
- Connect ECT to existing literature (complexity theory, logic, AI)

### 💻 Implementation
- Help build the ECT type system in SLIME
- Prototype the ECT-AI reasoning engine
- Build tooling for the research pipeline

### 📝 Documentation
- Improve formal definitions
- Write accessible explanations of ECT concepts
- Translate ideas into LaTeX for the preprint

## Process

1. Open an issue describing your proposed contribution
2. Fork the repo and create a branch
3. Submit a pull request with a clear description
4. All theoretical claims must include either a proof sketch or explicit open question status

## Standards

- All conjectures must be falsifiable or formally undecidable by construction
- No hand-waving — if something is informal, label it explicitly
- Attribution for all referenced work
